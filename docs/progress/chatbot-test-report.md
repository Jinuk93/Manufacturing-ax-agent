# 챗봇 기능 테스트 보고서

**테스트일:** 2026-03-17
**버전:** chat_agent.py v2 (DB + Neo4j + pgvector 3소스 통합)

---

## 개선 내용

챗봇(answer_chat)이 3가지 데이터 소스를 통합하여 답변을 생성하도록 개선:

| 소스 | 이전 | 이후 |
|------|------|------|
| pgvector 매뉴얼 검색 | ✅ | ✅ |
| DB 조회 (재고/작업/센서/정비) | ❌ | ✅ |
| Neo4j 온톨로지 순회 | ❌ | ✅ |

### 동작 흐름

```
사용자 질문
  ↓
1. 키워드 분석 → 필요한 DB 테이블 자동 선택
   (anomaly_scores, erp_inventory, mes_work_orders, maintenance_events)
  ↓
2. Neo4j 순회 → 고장코드가 언급되면 관련 부품/매뉴얼/정비이력 탐색
   (R4 REQUIRES, R5 DESCRIBED_BY, R3 EXPERIENCES, R8 RESOLVES)
  ↓
3. pgvector 의미 검색 → 질문과 가장 유사한 매뉴얼 청크 3건
  ↓
4. LLM(GPT-4o-mini)에 3소스 컨텍스트를 모두 전달 → 답변 생성
```
