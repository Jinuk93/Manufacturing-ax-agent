## 3. 판정 상세

### M1_CURRENT_PROGRAM_NUMBER

- 17개 실험이 전부 `{1}`만 사용, experiment_02만 `{0, 1, 4}` 사용
- 97.2%가 해당이면 "오류"가 아니라 **정상 동작**
- 해석: 0=대기/미실행, 1=메인 가공 프로그램, 4=보조 프로그램
- **결론: 필터링하지 않음**

### M1_CURRENT_FEEDRATE = 50

- 모든 공정 단계에서 고르게 발생 (17~33%)
- Repositioning(28.5%), End(28.4%)에서 상대적으로 높음
- 해석: 실제 절삭이 아닌 **공구 이동/대기 구간**의 기본 이송속도
- **결론: 오류 아닌 "비절삭 구간" 마커. F2 모델링 시 선택적 제외 옵션**

### X1_ActualPosition ≈ 198

- End 단계에서 29.9%로 가장 높고, Prep에서도 25.8%
- 해석: X축 **홈 포지션**(초기/복귀 위치)
- **결론: 오류 아닌 정상 위치. 필터 불필요**
